Aplicaciones móviles Desarrolladas en Android Studio (Java) Ver todas en Google Play

Lector de códigos de barras Ver en Google Play
Esta app. te permitirá realizar la lectura de códigos de barras (2 dimensiones), entregando la fuente con la cual fue generada y la longitud o largo del código leído.
Puzzle Mame Ver en Google Play
Soluciona puzzles con a las imágenes de los juegos de consola antiguos MAME (25 puzzles disponibles).
Código Morse Ver en Google Play
Esta aplicación te ayudara a aprender de forma fácil a utilizar el código o alfabeto morse. Ya sea a través de sonidos (beep), vibraciones o la reproducción a través de señales de luz.

Aplicaciones Windows Desarrolladas con Microsoft .NET (C#)

Respaldo
Aplicación para realizar copias de seguridad de tus archivos.
Esta aplicación permite a traves de la definición de tipos de archivos (extensiones) la posibilidad de realizar copias de seguridad (Respaldos).
Tag (Base de conocimientos)
Aplicación para generar una base de conocimientos a través de la creación de tips/tags (Notas).
Esta aplicación permite la creación de algunos tips o tags que permiten generar un base de conocimientos en base a caterogías y sub-categorías de distintos temas .
Control inventario
Registro y manejo de la cantidad de productos de una empresa.
Esta aplicación está pensada para llevar el registro del stock de los distintos productos a través de entradas y salidas. Permitiendo así tener un control detallado del stock actual.
Reproducción de voz
Permite la reproducción de voz a través de dos o más equipos conectados dentro de una red.
Esta aplicación puede ser utilizada dentro una red interna, permitiendo enviar y reproducir mensajes desde un equipo EMISOR (Cliente) hacia un equipo RECEPTOR (Servidor)
Nota: Dentro del equipo EMISOR (Cliente) se debe configurar la IP y puerto del equipo RECEPTOR (Servidor).
Mantenedor de Usuarios
Módulo para administrar los usuarios de un sistema.
Esta aplicación permite la administración de cuentas de usuarios, y perfiles dentro de un sistema u organización.
Menú principal
Panel de control centralizado de las aplicaciones de una empresa.
Esta aplicación permite integrar en un solo modulo los accesos a todas las aplicaciones que utilizas a diario (Sistemas propios de tu empresa, aplicaciones office, acceso al panel de control, impresoras, carpeta de descargas y otros). Además permite la administración de perfiles de usuario, según área y rol asignado.
Reporte de errores (Tickets)
Administración de ticket de atención.
Mediante esta aplicación los usuarios de la empresa, registran los problemas o incidencias que se van produciendo durante el día, ya sean problemas asociados al software/programas o si corresponde a problemas con el equipamiento/hardware. Esto alimenta un módulo de seguimiento que es monitoreado por los soportes del área informática, los cuales deben ir cerrando dichas incidencias.
Monitoreo de equipos
Esta aplicación permite un control de los computadores de la empresa.
Este programa provee la información de los equipos registrados dentro de un dominio, de tal forma que los soportes puedan tener información para actuar de forma preventiva ante situaciones de riesgo que puedan provocar la pérdida de información vital para la empresa, básicamente el monitor revisa el espacio en disco de un equipo, si su antivirus se encuentra actualizado, si sus respaldos están al día y otros. Además permite la conexión en tiempo real de equipo, para saber la información completa del equipo como por ejemplo: el nombre, su IP, MAC, sistema operativo, información de espacio en discos, impresoras conectadas, carpetas compartidas, procesos, servicios, software instalado y otros.

Aplicaciones Web Desarrolladas con ASP Clásico y Microsoft ASP.NET (C#)

Control de Procesos (Mecanizado y Distribución)
Proceso de control en la distribución de correspondencia.
Este sitio web permite a los usuarios, realizar distintos procesos operativos como por ejemplo: ingresar un proceso, generación de muestras, para finalmente ser impresas y distribuidas.
Control cotizaciones
Procesos internos de una empresa de climatización.
Este sitio permite la programación de visitas, realización de cotizaciones y control de las cotizaciones por facturar.
Control Terceros
Proceso de control en la distribución de correspondencia.
Este sitio web permite a los usuarios (terceros), generar todo tipos de consultas del proceso operativo que realizan, permitiendo así saber la trazabilidad completa de un envío, información de pagos, despachos, atención a clientes, entrega de envíos, solicitud de materiales y otros.
